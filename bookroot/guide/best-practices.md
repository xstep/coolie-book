# 目录结构
```
fullstack-project
├── node_modules ------------------- 后端使用的 node 模块
├── package.json ------------------- 后端使用的模块描述文件
├── webroot-dev -------------------- 前端开发环境根目录
│   ├── coolie.config.js ----------- 前端构建配置文件
│   ├── node_modules --------------- 前端使用的 node 模块
│   ├── package.json --------------- 前端使用的模块描述文件
│   └── static
│       ├── css
│       ├── img
│       └── js
│           ├── coolie-config.js --- 前端模块加载器配置文件
│           └── main --------------- 前端模块入口目录
├── webroot-pro -------------------- 前端生产环境根目录
└── webserver ---------------------- 后端根目录
```
