coolie —— 前端**构建工具**与前端**模块加载器**组合的工程化解决方案。


## 作为前端构建工具
- JS 文件的分析、合并、压缩、版本管理
- CSS 文件的分析、合并、压缩、版本管理
- HTML 文件分析、压缩、版本管理
- 入口模块的分析、分块构建、增量构建、合并、压缩、版本管理
- 静态资源的分析、压缩、版本管理


## 作为前端模块加载器
- 不需要提前编译
- 兼容 ie6 - ie11/chrome/firefox，以及各种手机浏览器
- 开发环境兼容 CommonJS、CMD 规范
- 支持加载任意的类型模块，如图片、样式、JSON 等
- 支持分块加载和异步加载
- 良好的构建工具配套支持
